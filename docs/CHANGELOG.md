# 更新日志

## [1.1.0] - 2024-03-19

### 🐛 Bug修复
- **修复游戏阶段跳跃问题**: 游戏不再从Pre-flop直接跳到Turn阶段
  - 添加 `playersActed` 数组追踪每个玩家是否已行动
  - 重写 `checkRoundComplete()` 函数确保所有玩家都有机会行动
  - 修复 `endBettingRound()` 中的玩家索引初始化问题

- **修复下注轮过早结束**: 确保每个玩家在每个阶段都能行动
  - 在所有玩家操作函数中添加行动标记
  - 加注时正确重置行动列表
  - 修复AI决策后的流程控制

### ✨ 新功能
- **改进AI决策逻辑**:
  - 添加位置因素考虑（后位更激进）
  - 实现底池赔率计算
  - 更智能的跟注/弃牌决策
  - 增加决策随机性避免可预测性

- **添加工具函数**:
  - `getNextActivePlayer()` - 获取下一个活跃玩家
  - `getActivePlayers()` - 获取所有活跃玩家
  - `resetPlayerBets()` - 重置玩家下注

- **全局错误处理**:
  - 添加 `window.onerror` 事件监听
  - 添加 `unhandledrejection` 处理
  - 提高游戏稳定性

### 🎨 样式改进
- **CSS兼容性修复**:
  - 添加标准 `appearance` 属性
  - 修复 `-webkit-appearance` 兼容性问题
